 Currency Converter Pro 💸
 
A modern, dark-themed currency converter built with *Python* and *CustomTkinter*.
 
 Features include:
* Real-time conversion logic.
* Sleek UI with Orbitron typography.
* Dropdown menus for easy selection.

 
 How to Run:
1. Install dependencies: pip install customtkinter
2. Run the app: python "Currency Converter App (final version).py"
